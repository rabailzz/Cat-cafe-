# The Cats Lounge

Welcome to **The Cats Lounge**, a cozy cat café website based in Melbourne that blends specialty coffee, comfort, and feline companionship.

---

## **Project Overview**

This project is a fully responsive, semantic web application built using HTML5, CSS3 (utilizing CSS Grid and Flexbox), and JavaScript (including ScrollReveal animations). It features a dedicated home introduction, an about section with a gallery of resident cats, a stylized menu board, and contact/location details.

---

## **Features**

* **Semantic HTML5 Structure**: Utilizes clear landmark elements like `<header>`, `<main>`, and `<footer>`, paired with an explicit `<h1>` heading strategy.


* **Modern CSS Layouts**: Built utilizing **CSS Grid** for macro page structures and **Flexbox** for micro component alignments.


* **Interactive Design**: Integrated with smooth scrolling, mobile navigation toggles, and scroll-triggered animations via ScrollReveal.js.
* **Accessibility & Performance Optimization**: Includes explicit image width and height dimensions to prevent Cumulative Layout Shift (CLS), along with SEO-friendly meta tags and screen-reader utility classes.



---

## **File Structure**

* `index.html`: The main entry point containing the webpage markup.
* `project1.css`: External stylesheet containing design variables, base styles, layout grids, and section rules.
* `project1.js`: JavaScript file managing mobile menu interactions, header scroll effects, and ScrollReveal animation configurations.
* `assets/css/styles.css`: Secondary external style rules.

---

## **Getting Started**

To view or run this project locally:

1. Clone or download this repository to your local machine.
2. Ensure all asset files (images, icons, and stylesheets) are kept in their respective paths relative to `index.html`.
3. Open `index.html` in any modern web browser.
